# Tools

#### 一些有用的工具文件、函数、插件等

#### hanzi_convert_pinyin.js

```js
/**
* 调用:
* hanzi_convert_pinyin("123，我爱你我的祖国！");
* 结果:
* "123，wǒ ài nǐ wǒ de zǔ guó ！"
* 调用:
* hanzi_convert_pinyin("123，我爱你我的祖国！", " ");
* 结果:
* "wǒ ài nǐ wǒ de zǔ guó"
*/
```

#### jianti_fanti_exchange.js

```js
/**
* 调用:
* jianti_fanti_exchange("广东");
* 结果:
* "廣東"
* 调用:
* jianti_fanti_exchange("廣東", false);
* 结果:
* "广东"
*/
```
